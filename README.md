# Computer Science Fall Semester Exhibition Website

## Overview

This project is a static exhibition website designed to showcase student-created music using Sonic Pi. The site provides a brief introduction to Sonic Pi and allows visitors to explore a collection of songs that replicate real-world music using code.

The website is intended for exhibition attendees of all ages, offering an interactive and accessible way to experience programming through music.

---

## Features

* **Dynamic Song Pages**
  Song content is loaded dynamically from a JSON file, allowing scalable and organized data management.

* **Genre-Based Navigation**
  Songs are grouped into genre-specific pages for easier browsing.

* **Custom Audio Player**
  Includes a custom-built audio player for:

  * Sonic Pi sample code playback
  * Student-created song playback

* **Code Display**
  Users can view the actual Sonic Pi code used to create each song.

* **Inactivity Timeout System**
  Automatically returns the site to the homepage after one minute of inactivity, making it suitable for public exhibition environments.

---

## Technologies Used

* HTML
* CSS
* JavaScript

This is a fully static website with no backend or external frameworks.

---

## Installation / Setup

1. Download or clone the repository.
2. Open `index.html` utilizing a live server to navigate.

No additional dependencies or installation steps are required.
